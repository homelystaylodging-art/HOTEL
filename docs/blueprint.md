# **App Name**: HomelyStay Suites

## Core Features:

- Informative Sections Display: Present all core content sections including Hero, About Us, Our Rooms, Amenities, and Contact details, with images integrated as specified.
- Dynamic Room Type Listings: Display various room categories with their unique descriptions and dedicated imagery from 'Guest Room Interior 1' and 'Guest Room Interior 2', with a sub-section for bathroom cleanliness from 'Thumbnail 5'.
- Direct WhatsApp Booking Call-to-Action: Implement a prominent 'Book via WhatsApp' button allowing immediate direct communication for inquiries and bookings.
- Interactive Location Map: Embed an interactive Google Maps component displaying the hotel's exact address, 'Near CEAT Road, Plot No 135 & 136, Ranjangaon Waluj MIDC, Chhatrapati Sambhaji Nagar'.
- Mobile-First Responsive Design: Ensure the entire website is fully responsive, optimized for fast loading and seamless display across all mobile, tablet, and desktop devices.
- AI-Powered FAQ Assistant: An AI tool providing instant answers to common guest questions about amenities, hygiene protocols, booking process, and local area information, enhancing user convenience and conversion.

## Style Guidelines:

- Color scheme: Light mode. Emphasizing cleanliness and approachability. The color palette focuses on warm, earthy, and trustworthy tones, evoking comfort and hygiene without sacrificing professionalism.
- Primary brand color: A refined, muted taupe for professionalism and warmth (HSL(40, 15%, 45%), converted to #827768). This forms the basis of textual elements and secondary components.
- Background color: A very light, desaturated cream for an inviting, clean canvas (HSL(40, 10%, 95%), converted to #F9F7F5).
- Accent color: A vibrant, encouraging orange to highlight Call-to-Action buttons and key interactive elements (HSL(10, 70%, 50%), converted to #D95C26). This provides a clear focal point.
- Headline font: 'PT Sans' (humanist sans-serif), offering a modern look with a touch of warmth for headlines, titles, and section headers.
- Body font: 'Inter' (grotesque-style sans-serif), chosen for its objective, neutral, and highly readable characteristics for all body text and descriptions.
- Utilize clean, modern line icons that communicate services and features efficiently, enhancing the website's professional and streamlined appearance without clutter.
- Adopt a mobile-first responsive layout, featuring generous whitespace for clarity, a single-column flow for mobile, and a clear, modular grid system for larger screens, ensuring ease of navigation and a 'streamlined flow'.
- Implement subtle, functional animations such as smooth transitions for navigation, gentle hover effects on interactive elements, and discreet fade-in or parallax scrolls for images, improving user engagement without impacting performance.
